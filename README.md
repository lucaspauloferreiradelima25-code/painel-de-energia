# ⚡ Painel de Energia

Um painel web moderno, interativo e responsivo desenvolvido para monitoramento e análise de consumo de diferentes fontes de energia. O projeto conta com gráficos dinâmicos para facilitar a visualização dos dados e uma interface limpa e intuitiva.

---

## 🚀 Tecnologias Utilizadas

* *React* (Biblioteca principal para construção da interface)
* *JavaScript (ES6+)* (Lógica de programação e manipulação de dados)
* *HTML5 & CSS3* (Estruturação e estilização moderna com foco em responsividade)
* *Vite* (Ferramenta de build rápida para o desenvolvimento)
* *Recharts / Chart.js* (Renderização de gráficos de consumo e fontes)
